## 使用說明

- **UML 路徑**：以 PlantUML 原始檔為準，可用 IDE 外掛或 [PlantUML](https://plantuml.com/) 產生圖片。
- **是否完成**：反映撰寫本表時之**計畫內完成度**；若你之後跑齊 block CV、Phase 3 或 Phase 4，請同步更新「成果」與「是否完成」欄。
- **與 STRUCTURE.md 之差異**：`docs/STRUCTURE.md` 描述目錄；本表專注於與 `研究方向.md` 的一對一對照。
