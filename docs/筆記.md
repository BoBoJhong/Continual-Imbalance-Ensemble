Old (舊資料模型)

意義： 最基礎的底線基準 (Baseline)。
作法： 模型僅使用較早年份的「歷史資料 (Historical data)」進行訓練，然後直接去預測未來的測試集資料 (Testing data)。
由來： 這是 3/3 會議中指導老師特別要求補上的基準線，用來了解如果不看任何新資料，模型的預測表現會有多差。
New (新資料模型)

意義： 原本計畫中的 Fine-tuning 基準線。
作法： 模型僅使用較近期的「新營運資料 (New operating data)」來進行訓練（或微調），並預測未來的測試集。
目標： 測試若只依賴距離測試集時間點最近的新資料，是否能預測得比較準確，但缺點是資料量可能較少。
O+N (Old + New / 舊+新資料模型)

意義： 原本計畫中的 Retraining 基準線。
作法： 將「歷史資料 (Old)」與「新營運資料 (New)」合併在一起，重新訓練整個模型，再預測測試集。
目標： 這是傳統機器學習最常見的作法，餵給模型所有能拿到的資料來更新模型。
