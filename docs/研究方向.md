# Class Imbalance (3)

• Continual learning with class imbalance (non-stationary datasets), e.g.
bankruptcy prediction (1999~2018) (Kaggle), stock prediction (學長的論文), and
time series medical datasets (UCI)
• 資料集切割: 例如 
a. 1999~2011 as the historical data, 2012~2014 as the new
operating data, and 2015~2018 for the testing data
 
b. 5-fold cross validation:1st+2nd folds as the historical data, 3rd + 4th folds as the new operating data, and the 5th fold for the testing data

• Baselines: 
1. Re-training the model using historical and new operating data
2.fine-tuning the model solely using new data

• Ensemble classifiers with/without dynamic classifier selection/dynamic ensemble
selection:

1. ‘Old’ models 1/2/3 (trained by historical data with under-/over-/hybrid
sampling)
2. ‘New’ models 4/5/6 (trained by new operating data with under-/over-/hybrid
sampling)
3. combination methods: 
a. two combined models: pairs of ‘Old’ and ‘New’
models
b. three combined models: two ‘Old’ + one ‘New’ or one ‘Old’ + two ‘New’
models
c. four combined models
d. five combined models
e. six combined models

• Study II: the effect of feature selection on the ensemble classifiers

## 繁體中文翻譯

# 類別不平衡 (3)

• 處理類別不平衡的持續學習（非平穩資料集），例如：破產預測（1999~2018）(Kaggle)、股票預測（學長的論文）、以及時間序列醫療資料集 (UCI)。

• 資料集切割：

- a. 以年份切割：例如將 1999~2011 作為歷史資料 (historical data)，2012~2014 作為新營運資料 (new operating data)，並將 2015~2018 作為測試資料 (testing data)。
- b. 5 折交叉驗證 (5-fold cross validation)：第 1 與第 2 折作為歷史資料，第 3 與第 4 折作為新營運資料，第 5 折作為測試資料。

• 基準方法 (Baselines)：

  1. 使用「歷史資料與新營運資料」合併重新訓練模型 (Re-training)。
  2. 僅使用「新資料」微調模型 (fine-tuning)。

• 具備或不具備「動態分類器選擇 (DCS) / 動態集成選擇 (DES)」的集成學習 (Ensemble classifiers)：

  1. 「舊 (Old)」模型 1 / 2 / 3（使用歷史資料，搭配欠採樣 / 過採樣 / 混合採樣來訓練）。
  2. 「新 (New)」模型 4 / 5 / 6（使用新營運資料，搭配欠採樣 / 過採樣 / 混合採樣來訓練）。
  3. 模型組合方法：
    - a. 兩個模型的組合：「舊」與「新」模型配對。
    - b. 三個模型的組合：兩個「舊」+ 一個「新」，或一個「舊」+ 兩個「新」模型。
    - c. 四個模型的組合。
    - d. 五個模型的組合。
    - e. 六個模型的組合。

• 研究二 (Study II)：探討特徵選擇對集成決策分類器的影響。
